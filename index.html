Projects

File Organizer Application GitHub

This Python-based application automatically organizes files into categorized folders based on file type. It uses the Watchdog library to monitor file system events and organize files in real time. Currently, I am implementing content-based classification using machine learning to further automate the categorization process, allowing the system to categorize files based on their content.

Languages & Libraries: Python, Watchdog, Pandas, Scikit-Learn

Data Assimilation Techniques GitHub

Implemented multiple data assimilation techniques, including Kalman Filtering, Extended Kalman Filtering (EKF), Particle Filtering, and Generalized Alpha-Beta Filtering for state estimation in dynamic systems.

Languages & Libraries: Python, Bayesian Filter

Movie Recommendation System GitHub

This project uses collaborative filtering and content-based filtering to recommend movies based on a user’s past ratings. I built an interactive recommendation engine using Python and various machine learning techniques, including matrix factorization. The system provides personalized movie recommendations based on IMDb data.

Languages & Libraries: Python, Pandas, Scikit-Learn, Flask, IMDb API

Super (9x9) Tic-Tac-Toe GitHub

A 9x9 Tic-Tac-Toe game with both single-player and two-player modes. The single-player mode features a bot that uses the minimax algorithm with alpha-beta pruning to play optimally. This project helped me sharpen my understanding of game theory and artificial intelligence algorithms.

Languages & Libraries: Python, Numpy

Neural Network for MNIST Digit Classification GitHub

I developed a neural network from scratch for the classification of MNIST digits using only NumPy. The project uses backpropagation and gradient descent to train the network, and I achieved 99.1% accuracy on the test dataset. This helped me gain a deeper understanding of neural network architectures and optimization techniques.

Languages & Libraries: Python, NumPy
